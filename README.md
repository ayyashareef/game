# 🎮 Kids Games

A little collection of browser games for ages 4–9. No install, no backend — just open `index.html`.

## Games
- 🚵 **Mountain Bike Adventure** (`bike.html`) — ride the hills, collect stars, 8 levels that unlock one by one.
- ✏️ **Thaana Letters / Haa Shaviyani** (`thaana.html`) — trace all 24 Thaana letters along a dotted guide line.
- 🔤 **ABC Letters** (`abc.html`) — trace A–Z (upper & lowercase) with fun example words.

The dashboard is `index.html`.

## Tracing & stars
Trace the letter, then tap **▶ Next** to be graded:
90%+ → ⭐⭐⭐, 50%+ → ⭐⭐, 40%+ → ⭐.

Progress is saved automatically in the browser (localStorage).

## Run locally
Open `index.html` in any modern browser.

## Hosting (GitHub Pages)
Push to GitHub, then in **Settings → Pages** set the source to the `main` branch / root.
The site will be served at `https://<your-username>.github.io/<repo>/`.
